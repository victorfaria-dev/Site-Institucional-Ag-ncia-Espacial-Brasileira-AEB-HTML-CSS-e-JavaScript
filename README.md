# Agência Espacial Brasileira (AEB)

Projeto acadêmico desenvolvido para a disciplina de Desenvolvimento Web, utilizando HTML5, CSS3 e JavaScript.

O site apresenta informações institucionais sobre a Agência Espacial Brasileira (AEB), incluindo sua história, missão, programas, localização e um formulário de contato com validação em JavaScript.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Layout institucional para desktop;
- Menu de navegação por seções;
- Banner principal;
- Seção sobre a AEB;
- Apresentação dos principais programas espaciais;
- Mapa da sede da Agência;
- Formulário de contato com validação em JavaScript;
- Botão de retorno ao topo.

## Objetivo

Aplicar os conhecimentos de desenvolvimento web por meio da criação de um site institucional estático, priorizando organização, acessibilidade e boas práticas de HTML, CSS e JavaScript.
